# JabalpurGo 🌍

**JabalpurGo** is a tourism web application that helps users explore famous places in Jabalpur.
The website provides information about tourist destinations, allows users to search places, view descriptions, and explore locations easily.

## 🚀 Live Website

https://jabalpur-go.vercel.app

## 📌 Features

* 🔍 Search for tourist places in Jabalpur
* 📖 View detailed information about places
* 🔊 Text-to-Speech description for accessibility
* 🌐 Hindi translation support
* 🗺️ View location using Google Maps
* 📍 Explore nearby tourist attractions

## 🛠️ Tech Stack

* **React.js**
* **Vite**
* **Tailwind CSS**
* **JavaScript**
* **Wikipedia API**
* **Google Maps Embed API


## 💻 Installation (Run Locally)
Clone the repository:

git clone https://github.com/Sanjanayadav25/jabalpurGo.git
Go to project folder:
cd jabalpurGo
Install dependencies:
npm install
Run the project:
npm run dev

## 🎯 Purpose of the Project

This project was built to practice **React development and API integration** while creating a useful tourism platform for exploring Jabalpur city.
